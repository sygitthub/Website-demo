## X-Sense 安室智能 · 官网示例

本项目为 X-Sense 安室智能的品牌官网示例页面，围绕“智慧安防 · 守护生命与健康安全”的定位，展示公司简介、核心产品、技术优势、全球布局与合作伙伴等信息。

### 功能与特点

- **品牌形象展示**：突显 X-Sense 作为面向欧洲与北美市场的智慧安防“出海”企业定位。
- **产品与方案**：概括家庭安防产品、智慧联动套装与公共项目解决方案的核心卖点。
- **技术护城河**：强调多传感器与无线技术、全球专利与国际认证优势。
- **全球化布局**：呈现“立足深圳、面向全球”的品牌战略与海外市场成绩。
- **动效与体验**：支持滚动淡入、数字自增、浮动卡片等丝滑动效，适配移动端。

### 使用方式

- **本地打开**：直接在浏览器中打开 `index.html` 即可预览页面效果。
- **通过 Node 服务预览（可选）**：
  1. 确保已安装 Node.js 环境；
  2. 在项目根目录执行：
     ```bash
     npm install -g serve
     serve .
     ```
  3. 在浏览器中访问终端输出的本地地址。

### 目录结构

- `index.html`：主页面结构与内容。
- `styles.css`：整体视觉样式与响应式布局。
- `main.js`：导航、滚动动效与数字自增逻辑。
- `package.json`：基础项目描述与简单本地预览脚本。
